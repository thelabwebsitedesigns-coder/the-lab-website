Your portfolio website, ready to deploy.

## OPTION 1: Deploy to Vercel (Easiest — 3 Minutes)

### Step 1: Create accounts
- Go to **github.com** and create a free account (if you don't have one)
- Go to **vercel.com** and sign up with your GitHub account

### Step 2: Upload this project to GitHub
1. On GitHub, click the **+** button in the top right → **New repository**
2. Name it `the-lab-website`
3. Keep it **Public** or **Private** (your choice)
4. Click **Create repository**
5. On the next page, click **"uploading an existing file"**
6. Drag and drop ALL the files and folders from this project into the upload area
7. Click **Commit changes**

### Step 3: Deploy on Vercel
1. Go to **vercel.com/new**
2. Click **Import** next to your `the-lab-website` repository
3. Click **Deploy**
4. Wait about 60 seconds
5. **Done!** Vercel gives you a live URL like `the-lab-website.vercel.app`

### Step 4: Add your custom domain (optional)
1. Buy a domain (like `thelabwebsitedesigns.com`) from Namecheap, GoDaddy, or Google Domains
2. In Vercel, go to your project → Settings → Domains
3. Add your custom domain and follow the DNS instructions
4. Your site is now live at your own URL

## OPTION 2: Deploy to Netlify (Also Easy)

1. Go to **netlify.com** and sign up
2. Click **"Add new site"** → **"Deploy manually"**
3. First, build the project locally (see below), then drag the `dist` folder into Netlify
4. Done! You get a `.netlify.app` URL

---

## Running Locally (For Development)

If you want to preview the site on your own computer first:

1. Make sure you have Node.js installed (download from nodejs.org)
2. Open a terminal in this project folder
3. Run: npm install
4. Run: npm run dev
5. Open the URL it shows (usually http://localhost:5173)

To build for production: npm run build
This creates a `dist` folder with your final site files.
